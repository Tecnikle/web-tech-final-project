# E-Portfolio Website

## Project Overview

The E-Portfolio Website is a personal portfolio developed to showcase my background, technical skills, education, projects, and career goals. The website serves as a central location where visitors can learn more about me, and view my accomplishments
This project demonstrates my knowledge of front-end web development using HTML, CSS, and JavaScript while emphasizing responsive design, accessibility, and clean user interface principles.

---

## Project Objectives

The primary objectives of this project were to:

- Design and develop a professional personal portfolio website.
- Practice HTML5 semantic structure and accessibility.
- Apply CSS styling to create a modern and visually appealing interface.
- Implement JavaScript to enhance user interaction.
- Organize information into easy-to-navigate sections.

---

## Features

- **Landing Page** webpage that serves as the landing site
- **About Me page** webpage going into more depth about me
- **Capstone Project Page** webpage showing off my senior capstone project
- **Junior Project Page** webpage going over my junior data science project
- **R Published Document Page** webpage created via RStudio's publishing system
- **Webpage for this project** webpage for discussing this project
- **Final Webpage as closer** webpage meant to show off the remaining skills that might not have fit on another page
