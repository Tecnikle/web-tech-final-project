Data Set and Why I Chose It

Link to Dataset
My dataset contains the name, platform, release year, publisher, genre, and sales split across North America, Europe, and Japan.

I'm interested in this topic because I love games and have been gaming since I was a kid. My goal with this dataset is to map sales trends within series (Call of Duty, Battlefield, Super Mario, etc.) as well as overall yearly sales trends in order to see how consumer tastes shift from year to year.

For example, did the sales of one game within a series increase because it was more well received, or because overall game sales increased that year?

Additional Info

I have many hypotheses that I want to test throughout this project. This is important because it allows me to continually discover new insights as different assignments are given, rather than repeating the same findings.
